Make sure python and all relevant libraries are installed: numpy

Run Vybe.py using python3 Vybe.py

The main menu will bring up 4 options:
    1. change emotions
    2. 7 day report
    3. current day report
    4. exit

Select the option you would like by entering the corresponding number and pressing enter
Follow the instructions prompted by the program for inputting emotions, changing and getting reports
